jukebox settings :)

1. to change volume, open tags.json and edit 'volume' — number from 1 to 10

2. to add new songs —

a) download songs
b) convert to 11kHz (11025Hz), mono, 16 bits per sample, .wav file
c) rename as needed, add into sd card
d) scan nfc tag to obtain nfc id
e) add entry in tags.json with nfc id and file name

that is all :)
